# **App Name**: AgriGenius

## Core Features:

- Farmer Profile Creation: Allow farmers to create a profile with location, land size, and basic farm details.
- AI Crop Recommendation: Suggest the best crops for the upcoming season based on soil data, weather forecasts, crop rotation history, and market trends, providing expected yield, profit margin, and sustainability score. Uses the Gemini API.
- AI Crop Profit Predictor: Calculate and display expected yield, income, expenses, and final profit/loss estimate for a chosen crop based on land size, selling price, and input costs. Uses the Gemini API.
- AI Disease Detection: Detect crop diseases or nutrient deficiencies from uploaded images using AI (Gemini API), providing explanations and recommended actions, along with optional visual highlights and a severity meter. The disease detection feature may use reasoning to decide whether to make use of past weather data as a tool.
- AI Chat & Voice Assistant: Integrate Gemini API for text chat and voice-based Q&A, allowing farmers to ask questions and receive clear, natural language responses with actionable steps. Multilingual support is provided for English, Telugu, and Hindi, including voice reply playback if possible.
- Offline Mode & Sync: Enable offline access to key screens and static content, storing user inputs locally and automatically syncing data when the device reconnects to the internet.
- Push Notifications: Send push notifications and in-app alerts for weather warnings, market price changes, and crop-care reminders.

## Style Guidelines:

- Primary color: Saturated cyan (#4DD0E1), evoking cleanliness and scientific precision.
- Background color: Light cyan (#E0F7FA), for a calm and airy feel.
- Accent color: Analogous blue-green (#4CAF50), for action items and positive indicators.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern look with some warmth.
- Use simple, clear icons suitable for low-literacy users, representing crops, weather conditions, and chat functions.
- Mobile-first design with a clean, uncluttered layout. Use of cards to present key information, and clear, prominent buttons for key actions.
- Subtle animations for loading indicators and transitions to provide a smooth user experience even with slow internet connections.
